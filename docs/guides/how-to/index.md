# How To

Welcome to the How To section. Here you'll find practical, step-by-step guides for common tasks when working with Lion Web Components.

Whether you're just getting started with Lion, building your first form, extending existing components, or creating custom fields, this section provides clear instructions and best practices to help you succeed. Each guide is designed to walk you through the process with real-world examples and code samples.

**Browse the guides below to:**

- Set up your development environment and get started with Lion
- Learn how to create accessible and well-structured forms
- Discover how to extend native input components
- Build custom form fields tailored to your needs
- Follow proven patterns and best practices used by the Lion community

If you can't find what you're looking for, check out our [Fundamentals](../../fundamentals/) section for deeper conceptual knowledge, or visit our [Components](../../components/) documentation for detailed API references.
