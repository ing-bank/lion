Why Lion?

White Label

Style the way you need it - no need to override any opinionated existing styling.

Accessible

All of our components align to platform standards, WAI ARIA patterns, and are always tested on accessibility, both through aXe automated tests but also manually with screen readers.

Extendable

Our components are meant to be extended and allow for flexible integrations.

Fast & Small

Using LitElement as a base class makes us small and fast.

Design System

Lion is an ideal fundament for building a Design System.

Reuse Documentation

Not only the code but also the documentation can be reused and extended.
